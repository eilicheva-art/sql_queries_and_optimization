# ✈️ SQL-задачи для анализа авиаперевозок (Demo Air)

Этот репозиторий содержит решения SQL-задач, выполненных в рамках учебного курса DA лаборатории Aston. В качестве источника данных используется демонстрационная база данных авиакомпании (схема `prod`), содержащая информацию о самолётах, аэропортах, рейсах, билетах и бронированиях.

## 📁 Структура репозитория

- `Avio_week_1.sql` – базовые SELECT, фильтрация, сортировка, агрегатные функции, работа с JSON.
- `Avio_week_2.sql` – JOIN, UNION, подзапросы, CASE, работа с датами, оконные функции и др.
- `Avio_week_3.sql` – подзапросы, обобщенные табличные выражения.
- `Avio_week_4.sql` – оконные функции.
- `Avio_week_5.sql` – представления.
- `Avio_week_6.sql` – функции.
- `Avio_week_7.sql` – оптимизация.
- `README.md` – этот файл.

## 📊 Используемые таблицы

В базе данных `prod` присутствуют следующие таблицы:

- `airplanes_data` – данные о самолётах (модель, дальность, скорость и т.д.)
- `airports_data` – справочник аэропортов (код, название, город, часовой пояс)
- `boarding_passes` – посадочные талоны
- `bookings` – бронирования (сумма, дата)
- `flights` – рейсы (статус, расписание)
- `routes` – маршруты (аэропорты вылета/прилёта, продолжительность, дни недели)
- `seats` – места в самолётах (класс обслуживания)
- `segments` – сегменты перелётов (цена, класс)
- `tickets` – билеты (пассажир, направление)
- `timetable` – представление с локальным временем вылета
